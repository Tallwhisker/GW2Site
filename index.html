About GuildWars 2 Inventory Viewer

I wanted a way to see the inventory of my account and characters outside of the game, so I decided to create this unofficial webapp.


This website is entirely client-side and stores all data in your browsers localStorage. I provide a base set of data that will be used to reduce the amount of API requests. If you wish to remove all stored data, this is possible via the menu.


An API key from your GW2 account is required with the permissions:

  • account
  • inventories
  • characters

Character data sometimes take a long time to download, and they do occasionally fail. If you see some icons of Spaghetti named "Unknown Item" it's because the item has no data yet. It should refresh and fix itself after a few seconds.
