About GuildWars 2 Inventory Viewer

I wanted a way to see the inventory of my account and characters outside of the game, so I decided to make a tool to do so.


This website is entirely client-side and stores all data in your browsers localStorage. This means that the website host has absolutely 0 of your data. It's all contained in your browser.


An API key from your GW2 account is required with the permissions:

  • account
  • inventories
  • characters

Due to the character requests being slow, it takes a bit to retrieve the data. They also fail regularly due to unknown reasons. After any new items have been found, they will show with temporary icons. When all modules are done downloading, they should refresh themselves.

If you see some icons of Spaghetti named "Unknown Item" this is because the GW2 API did not return any data for that item.
